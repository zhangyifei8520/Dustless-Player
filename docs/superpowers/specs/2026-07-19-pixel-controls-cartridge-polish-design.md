# 像素控件与卡带视觉校准设计

## 目标

在不改变主机素材、播放状态机和桌面单屏构图的前提下，统一像素字体，修正控制按钮、卡槽提示、卡带尺寸及全屏视频适配。

## 字体

- 使用用户提供的 `FZG_CN.ttf`，字体内部名称为 `PoxiaoPixel`。
- 字体作为本地站点资源加载，不依赖在线字体服务。
- 右上角 `首页 / 收藏库 / 关于` 使用 `PoxiaoPixel`，白色文字和黑色硬边像素描边。
- `POWER / 全屏 / 播放` 使用 `PoxiaoPixel`，黑色文字且不加描边。

## 主机控制区

- Power 与全屏按钮取消 hover/focus 的外侧白色虚线框。
- 键盘焦点改用按钮内部高亮，不增加外部尺寸。
- 两个按钮使用同一尺寸和同一垂直坐标。
- 按钮相对当前略微放大，内部 Power 和展开图标缩小至不会接触边框。
- `把卡带拖进插槽 · 或点击播放` 移入白色控制面板，固定在蓝色卡槽虚线框正下方并水平居中。
- 删除主机底部原有的独立提示行。

## 顶部导航

- 导航文字、图标、间距整体缩小至当前约 50%。
- 导航继续靠页面右上角，不改变按钮顺序和占位交互。
- 文字和 CSS 图标保持白色、黑色像素描边。

## 三张卡带

- 仅参考 `Library.dc.html` 中的卡片语言：三色外壳、3px 黑色描边、米白内容面板、方形硬阴影、顶部缺口、底部状态与日期。
- 三张卡带连同播放条组成的整体相对当前缩小约 20%，三列在紫色柜体内水平居中。
- 白色播放条相对卡带本体更宽、更高；在缩小后的整体中保持清晰的主操作层级。
- 播放条不越出紫色柜体，卡片悬停仍上移且不出现白色虚线框。

## 全屏播放

- 不增加静音按钮。
- 进入全屏时卸载主机显示屏内的播放器，只挂载全屏播放器，任何时刻最多存在一个 iframe。
- 全屏播放器保持 16:9，宽度同时受 `92vw` 和可用视口高度约束。
- 使用动态视口高度，确保播放器、关闭按钮和返回提示在同一窗口内完整可见，不产生页面滚动。

## 验收

- 1440×900 与 1920×1080 下页面主体保持单屏展示。
- 两个主机按钮尺寸一致、垂直对齐、图标不溢出且没有外侧虚线。
- 导航约为当前的一半尺寸并正确使用本地像素中文字体。
- 卡槽提示位于蓝色虚线框正下方。
- 三张卡带及播放条整体缩小约 20%，居中且不越出柜体。
- 全屏视频完整显示并保持 16:9；小播放器停止且 iframe 总数为一。
