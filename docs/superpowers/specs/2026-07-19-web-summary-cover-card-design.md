# 网页摘要封面卡设计

## 目标

当普通网页无法在显示器内完整阅读时，沿用现有深色电子风摘要卡，并让右侧出现一张倾斜的封面小卡，使降级状态仍像一个完整的可爱像素界面。

## 布局

- 左侧保留 `SAVED SUMMARY`、无法读取说明、收藏标题、收藏库原始摘要和原链接按钮。
- 右侧放置米白色封面小卡，带深色像素描边、右下偏移阴影和轻微顺时针倾斜。
- 若 Reader 已取得文章首图，封面卡展示该图片；没有首图时展示既有像素占位图。
- 不改变视频播放器、卡带、电视机或全屏交互。

## 数据与降级

- 阅读器接口在无法提取正文时，仍尽力返回可用的首图；若不存在首图，客户端使用占位图。
- 摘要文字始终使用收藏库记录的 `title` 与 `summary`，不虚构网页内容。

## 验收

- 正常可读网页保持既有图文阅读小窗。
- 不可读网页显示完整摘要卡，标题、摘要、原链接均可见。
- 有图片时，右侧小卡显示首图；无图时显示占位。
- 小卡不会遮挡摘要文字或原链接按钮。
